# Playwright Login Automation Framework with Allure Reporting

A robust End-to-End (E2E) test automation framework built using JavaScript and Playwright. This framework automates user authentication workflows and validates critical login scenarios across modern web applications.

## Features

* End-to-End login automation testing
* Built with Playwright and JavaScript
* Page Object Model (POM) design pattern
* Cross-browser execution (Chromium, Firefox, WebKit)
* Data-driven test support
* Screenshot capture on test failures
* Detailed Allure Reports with execution history
* Parallel test execution
* CI/CD ready for GitHub Actions and Jenkins
* Reusable utilities and custom helpers

## Test Coverage

* Valid user login
* Invalid credentials validation
* Empty username/password validation
* Session management verification
* Logout functionality
* UI element validations
* Error message verification

## Reporting

The framework integrates Allure Reports to provide:

* Interactive test dashboards
* Pass/Fail statistics
* Execution trends and history
* Detailed test steps
* Screenshots for failed tests
* Logs and attachments
* Failure analysis and debugging insights

## Technology Stack

* JavaScript (ES6+)
* Playwright
* Node.js
* Allure Report
* GitHub
* GitHub Actions / Jenkins

## Execution Flow

1. Execute Playwright test suites
2. Generate Allure result files
3. Build Allure HTML reports
4. Review execution metrics and test results

This framework is designed to be scalable, maintainable, and suitable for enterprise-level web application testing.
